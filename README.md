# Automotive-Warranty-Claims-Analytics
End-to-End Automotive Extended Warranty Claims Analytics using Excel &amp; Power BI

📌 Project Overview

This project focuses on analyzing Automotive Extended Warranty Claims Data to evaluate claim costs, approval trends, dealer performance, and regional concentration risk.

The solution was developed using Microsoft Excel for data preprocessing and Microsoft Power BI for interactive dashboard creation and business insights generation.

The objective is to support data-driven decision-making and optimize warranty cost management.

🎯 Business Objective

Analyze total claim cost impact

Evaluate warranty approval & rejection patterns

Identify top and low-performing dealers

Examine region-wise business contribution

Track monthly claim growth trends

Detect operational bottlenecks

Identify business concentration risk
